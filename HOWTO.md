# Manuel d'utilisation #

## Utilisation du programme via un terminal ##

Etapes : 
1. Ouvrir le projet dans un terminal
2. Compiler les classes en entrant la commande : javac *.java
3. Lancer le programme en entrant la commande : java Analyzer

## Utilisation du programme via l'interface ##

Etapes :  
1. Lancer le fichier .jar
2. Cliquer le bouton "Select Input Text File" pour choisir le fichier trace
3. Cliquer le bouton "Analyze" pour afficher le résultat de l'analyse de la trace
4. Cliquer le bouton "Select Output Text File" pour sauvegarder le résultat de l'analyse dans un fichier
5. Cliquer le bouton "Save" pour sauvegarder le résultat de l'analyse dans un fichier

OU
1. Ouvrir le projet dans un terminal
2. Compiler les classes en entrant la commande : javac *.java
